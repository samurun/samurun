## Overview

เป็นแพลตฟอร์มจัดการทรัพย์สิน (ที่ดิน, สิ่งก่อสร้าง, NFT, การจัดการประตู) เพื่อให้ผู้ใช้สามารถจัดสรรทรัพยากรต่างๆ ได้สะดวกผ่านเว็บไซต์

## Features

- Authentication (bitkubnext)
- Map
  - switch area (bitkub city, mutelulu)
  - filter land and buildings by sizes (X,S,M,L,XL)
  - filter Type (Land, Living house, Concession, My Land)
  - search Concession or Living house
  - zoom in and out
- Concession
  - add, remove NFT in concession (from bitkub chain)
  - door setting
  - show floor plan

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn
- viem.sh
- Leaflet.js
- NextAuth.js

## Additional Information

- User-friendly interface for easy navigation
- Integration with blockchain for secure transactions
